# Experiment Protocol Example

This is a public-safe template for writing down the question before running a
research experiment. Keep real universes, real parameters, run IDs, and
conclusions in a private workspace.

## Experiment

- experiment_id: synthetic_qvm_example
- question: Does a generic weighted factor score behave as expected on a
  synthetic fixture?
- hypothesis: Synthetic ranks should move when illustrative weights change.
- phase: EXPLORATORY

## Universe

- universe definition: synthetic common-stock fixture only
- in-sample period: 2026-01-01 to 2026-03-31
- out-of-sample period: not applicable for this public example

## Plan

- max planned variants: 3
- primary metric: after-cost return
- secondary metrics: after-tax return, benchmark return, max drawdown, cash
  drag, turnover
- reject conditions: pipeline errors, missing required factor groups, or
  synthetic expectations not met
- paper-test entry conditions: not applicable for this public example

## Notes

- This template is a research planning note, not an approval or compliance
  artifact.
- Public examples must not contain real strategy conclusions or private
  parameter choices.
